# UClaim: A Digital Lost & Found Reporting and Claim Management System for Universities

A web-based Lost & Found platform built for university campuses. Students can report lost or found items, track claim statuses, and coordinate item recovery through the Student Affairs Office (SAO).

## 🔗 Live Demo
[uclaim-sable.vercel.app](https://uclaim-sable.vercel.app)

## 🛠️ Tech Stack
- **Frontend:** React.js, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Deployment:** Vercel

## ✨ Features
- Report lost or found items with photos
- Search and filter items by category, status, and date
- Submit claims with proof of ownership
- Staff review and approval workflow
- Real-time claim status tracking
- Email notifications
- Admin dashboard for full system management
- Accessibility settings (dark mode, dyslexia font, color blind mode)

## 👤 Roles
- **Student** — Report, search, and claim items
- **Staff** — Review and manage claims at the SAO
- **Admin** — Manage users, settings, and reports
